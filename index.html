About the Console Screen and Visual Studio

Visual Studio is an coding IDE. I am using it right now! You have to first install it on your device. Then you have to follow the steps it shows to set it up on your device. After you set it up accroding to you liking, you can start coding! Create a folder, open it up on VS (Visual Studio) and create something awesome!! :)

The console sreen is a tool coders use. Console is generally known as the output screen in programming where we can compile and execute programs by giving commands. We can give inputs and see the outputs. We can also log stuff using the console.log and open it up in the console screen. We can do that by right-clicking on our mouse, then clicking on Inspect and going to the console screen at the top. Go ahead and try it now! See what is there!! :)
